# AntiPages — Template Contract

## Brief Document Description

Standard document that defines the structural, semantic, and technical architecture of AntiPages.

**Its function is to:**
* Define mandatory blocks.
* Define the H1 pattern and metadata.
* Define irreversibility rules and CTAs.
* Define allowed variables.
* Prohibit visual flags or structural exceptions.
* Establish the relationship between:
    * Single parametric component.
    * JSON dataset.
    * Global visual system.

**This document:**
* Does not contain final content.
* Does not contain multiple layouts.
* Does not define visual variants.
* Does not allow customization per landing page.
* Does not introduce a CMS.

**It is the contract that guarantees:**
* Structural uniformity.
* Statistical comparability.
* Industrial reproducibility.
